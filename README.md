# Health-Data-Analysis-Power-BI-
Designed an interactive Power BI dashboard to analyze healthcare data, including patient demographics, disease distribution, treatment outcomes, and hospital performance. Used Power Query for data cleaning and created KPIs, charts, and slicers to provide actionable insights for healthcare management.


This project presents an interactive Power BI dashboard built using a healthcare dataset containing over 55,000 patient records. The dashboard helps analyze patient demographics, hospital admissions, medical conditions, blood groups, medications, and test results through dynamic visualizations and KPIs.

Key Features
📊 Interactive Power BI Dashboard
👨‍⚕️ Analysis of 55,000+ patient records
🏥 Hospital-wise and city/state-wise analysis
📈 Medical condition distribution
🩸 Blood group analysis
💊 Medication analysis
🧪 Test result analysis (Normal, Abnormal, Inconclusive)
📅 Admission and discharge trend analysis
🎯 Interactive filters and slicers
